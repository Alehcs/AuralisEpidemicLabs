# Auralis Epidemic Labs Frontend

React, TypeScript, and Vite interface for creating, stepping, and inspecting
the minimal deterministic simulation.

```bash
npm install
npm run dev
```

Vite proxies API paths to `http://127.0.0.1:8000` during development. Set
`VITE_API_URL` when the backend is hosted elsewhere.

Validate production output with `npm run typecheck` and `npm run build`.
